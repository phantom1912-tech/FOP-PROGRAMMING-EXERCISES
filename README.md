# Foundations of Programming – Assignments

## 📌 Overview
This repository contains my assignments for the **Foundations of Programming** course (B.Tech – Computer Engineering). Each assignment demonstrates problem‑solving using **C programming**, along with structured write‑ups including **problem statements, objectives, theory, algorithms, programs, sample outputs, conclusions, and FAQs**.

---

## 🎯 Objectives
- To strengthen understanding of **basic programming concepts**.  
- To apply **decision control structures** (if, if‑else, switch).  
- To practice **loops and iterations** (for, while, do‑while).  
- To develop skills in **algorithm design and flowchart representation**.  
- To connect theory with practical coding exercises.

---

## 🛠️ Platform
- Language: **C**  
- Compiler: GCC (Linux environment)  
- IDE/Editor: Any text editor or IDE (e.g., VS Code, Code::Blocks)

---

## 📖 Structure of Each Assignment
Every assignment folder includes:
1. **Problem Statement** – What the program is supposed to do.  
2. **Objective** – Learning goals.  
3. **Theory** – Background concepts.  
4. **Algorithm** – Step‑by‑step logic.  
5. **Program** – C source code.  
6. **Sample Input/Output** – Example runs.  
7. **Conclusion** – Key takeaways.  
8. **FAQs** – Common questions and clarifications.

---

## 🚀 How to Run
1. Clone the repository:  
   ```bash
   git clone https://github.com/yourusername/foundations-of-programming.git
   ```
2. Navigate to the assignment folder:  
   ```bash
   cd Assignment-01
   ```
3. Compile the program:  
   ```bash
   gcc program.c -o program
   ```
4. Run the executable:  
   ```bash
   ./program
